# Cthulhu Rules extension for Inform7

This Extension add pg skills, combact, sanity and mental point. 

How use it
 -----------
 Create a folder(called ZARDOZ) in /your/path_to/Inform/Extensions/ & put the file inside.
 
 Import extension at the top of your source file  

Example:

```
"storyname" by "authorname".
Include the Standard Rules by Graham Nelson.
Include Cthulhu Rules by ZARDOZ.

Other code here
```
